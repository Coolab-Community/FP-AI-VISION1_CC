Purpose

STM32_Fs is a library of functions providing abstraction for file management (on top of FatFs) over SD interface.

Update History

Main Changes

Maintenance release

New features:

  • Add STM23Fs_GetImageInfoBMP() to read bitmap image information.
  • Add STM32Fs_CreateDir() to create a directory on the filesystem.
  • Add STM32Fs_ReadImageBMP() to read pixel values from bitmap file.
  • Add STM32Fs_WriteImageBMP16() to write RGB565 bitmap images.
  • Add STM32Fs_WriteImageBMPGray() to store 8-bit grayscale bitmap images.

Bug fixes and other changes:

  • Change package license to MIT License
  • Wrap macro statements with do-while-zero constructs
