Purpose

STM32H747I-DISCO Applications contains AI based application examples.

Update History

Main Changes

  • Adds STM32 USB Webcam application.
  • Adds B-CAMS-OMV camera bundle support.
  • Deprecates STM32F4DIS-CAM (OV9655) support. Use B-CAMS-OMV camera bundle (OV5640) instead.
  • Improves the camera capture data acquisition flow with internal line buffer to support higher pixel clocks.
  • Updates X-CUBE-AI to 5.2.0.
  • Changes SD card bus width to 1-bit to avoid conflicts with camera pins.

Main Changes

  • Adding two Person Presence Detetcion applications: one based on a low complexity MobileNetV1 0.25 model and one based on a medium complexity MobileNetV2 0.35 model.
  • Reworking the overall software architecture.
  • Further optimizing the internal SRAM memory occupation for the “Full internal” memory allocation scheme (up to 30% gain versus previous release).
  • Float and Quantized C models generated by X-CUBE-AI V5.1.2 tool.

Main Changes

  • Adding library of functions enabling “on-board” validation and debug capabilities as well as camera frame capture (for dataset collection purpose).
  • Adding support for file management on top of FatFS over SD interface.
  • Adding support for NN model quantized with Integer Arithmetic method.
  • Optimizing the internal SRAM memory footprint of the “Full internal” configurations for the Quantized models.
  • Float and Quantized C models generated by X-CUBE-AI V5.0.0 tool.

Main Changes

  • With a view to enable proper startup of Cortex-M4 core:
    • Adding a startup_cm4.s file (to each application project supported) containing a simple vectors table with reset handler looping in an infinite loop.
    • Modifying the linker files (for each application project supported) in order to place the Cortex-M4 vectors table at adress 0x08100000.

Main Changes

First official release of STM32H747I-DISCO Applications

Contents

  • Image classification example based on Food recognition neural network.
  • Demonstrates integration of Float and Quantized C model generated by X-CUBE-AI V4.0.0 tool.
  • Includes support of several data memory layout so to fit with user requirement.
